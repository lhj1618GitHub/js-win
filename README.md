# windows浏览器脚本

#### win浏览器脚本：

自动跳过中转页；悬停预览放大图片；搜索引擎切换器；视频速度控制；

哔哩哔哩-免登录最高画质；自动点击记录器;书签同步助手(Gitee Gists)；


#### 安装教程

1.  方式一：下载后直接导入篡改猴类管理器。[暴力猴](https://violentmonkey.github.io/)-[GitHub](https://github.com/violentmonkey/violentmonkey/releases) / [篡改猴](https://www.tampermonkey.net/) / [脚本猫](https://scriptcat.org/zh-CN)-[GitHub](https://github.com/scriptscat/scriptcat)
2.  方式二：下载后用记事本打开，复制全部内容，覆盖粘贴到篡改猴空白脚本内。
3. 同步[Gitee](https://gitee.com/lhj1618/js-win)

#### 使用说明

1.  图片原图查看器：鼠标悬停图片放大预览，默认悬停延时2s；
2.  自动跳过网页中转页：遇到不能自动跳转的中转页时，在中转页页面，菜单选择***一键添加当前页面规则***，可自动添加匹配规则；
3.  搜索引擎切换：可一键多引擎同时搜索，在搜索引擎页面的顶部居中显示悬浮图标，光标触发面板，自动提取关键词快捷搜索，可管理搜索引擎；非搜索引擎页面可在篡改猴菜单中选择临时显示；
4.  视频速度控制器：在视频框左上角显示速度指示器，点击或右键可调整倍速；双击重置到1.5倍。
5.  Bilibili-不登录-最高画质：无限试用最高画质-无需登录，会自动退出账号；
6.  自动点击记录器：记录在网页中点击的元素，下次访问自动点击；
7.  书签同步助手 (Gitee Gists)：同步书签到Gitee的gists,需token；可上传/下载html格式书签文件；

**注：此处所有脚本内均无更新链接**
